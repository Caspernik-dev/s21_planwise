# Дизайн: полноэкранный режим показа сценария (критерий 5)

Дата: 2026-05-27. Закрывает пункт 5 `docs/requirements-compliance.md` (проекторный/презентационный режим показа сценария на экране в классе).

## Проблема

Нет режима показа сценария на проекторе. Учителю негде вывести занятие крупно на экран ученикам.

## Решение

Клиентский overlay-компонент поверх редактора. Без БД, без server actions, без миграций — чистая презентационная обёртка над `ScenarioContent`.

## Слайды (гранулярность — по этапам, контент — тезисы)

Чистый хелпер `buildSlides(content, meta): Slide[]` (TDD):

- **Титульный слайд** — `title` крупно + бейджи (`direction`, `formatGrade(grade)`, `durationMin мин`, `format`).
- **По одному слайду на этап** — заголовок этапа + хронометраж; по каждой активности:
  - метка типа (Беседа/Квиз/Игра/Задание/Видео) как подзаголовок;
  - если у активности есть `questions` (непустой массив) — показываем их крупными буллетами;
  - иначе — показываем `text` активности (инструкция к действию).

Тип `Slide`:
```ts
type Slide =
  | { kind: 'title'; title: string; badges: string[] }
  | { kind: 'stage'; title: string; durationMin: number; blocks: { typeLabel: string; questions?: string[]; text?: string }[] }
```

## UI (`components/scenario/PresentationMode.tsx`)

- Кнопка «Показ» в шапке редактора (рядом с PDF/DOCX/История) открывает overlay, работающий поверх **текущего** `content` редактора (несохранённые правки видны).
- Запрос браузерного Fullscreen API на контейнере (best-effort: если отклонён/недоступен — overlay всё равно на весь viewport).
- Навигация: ← / → / Space — листать, Esc — выход; экранные кнопки «Назад»/«Далее» + счётчик «N / M».
- Крупный шрифт, центрированная колонка, фирменная палитра brand/neutral, высокий контраст для проектора.
- Длинный `text` активности (например «Учитель: …») — прокрутка внутри слайда, не обрезаем.
- Выход: ✕ и Esc → закрыть overlay + выйти из fullscreen.

## Тесты

- `buildSlides` — юнит-тесты (TDD): титульный слайд, слайд на этап, правило «questions есть → буллеты, иначе text», метки типов, бейджи.
- Overlay-компонент — без юнит-тестов (DOM/fullscreen/клавиши). Гейты: tsc, biome, build.

## Вне scope (YAGNI)

Показ на публичной странице `/s/[token]`, заметки докладчика, таймер этапа, анимации переходов.
